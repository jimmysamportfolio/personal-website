#### OVERVIEW

# Routing tasks to the right agent, every time.

As multi-agent systems grow in complexity, one of the hardest problems becomes classification — figuring out which specialized agent should handle a given task.

Agent-Router is an eval-driven multi-agent classification framework that intelligently routes incoming requests to the most appropriate agent based on the task's characteristics.

## A framework built for composability.

The system uses evaluation-driven routing to continuously improve classification accuracy. Rather than relying on static rules, Agent-Router learns from outcomes to make better routing decisions over time.

#### IMPACT

Agent-Router provides a reusable foundation for building sophisticated multi-agent systems where task delegation is critical to performance and reliability.
